# Go CRUD API with MySQL

A simple RESTful CRUD API built with Go, Gorilla Mux, and MySQL. Supports creating, reading, updating, and deleting users with proper error handling and HTTP status codes.

---

## Table of Contents

- [Features](#features)
- [Prerequisites](#prerequisites)

---

## Features

- Full CRUD operations (Create, Read, Update, Delete)
- MySQL database integration
- Proper HTTP status codes and error handling
- JSON request/response format
- Input validation
- Password field excluded from GET responses

---

## Prerequisites

- [Go](https://golang.org/dl/) (1.18+)
- [Docker](https://www.docker.com/) & Docker Compose
- [MySQL](https://www.mysql.com/) (via Docker)
